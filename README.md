# movie-repertoire

Simple website that shows movie repertiore for random city/area.
This project is created using simple and basic technologies like: 
- Node.js + Express.js (BE)
- HTML5 + Bootstrap + jQuery + AJAX (FE)
- SQLite 3 (Database)
